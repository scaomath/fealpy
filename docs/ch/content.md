# FEALPy: Finite Element Analysis Library in Python 

FEALPy 是一款偏微分方程数值解的开源算法库，以对象化和数组化的编程技术实现，简单
易用且易维护扩展，已集成了丰富的网格数据结构和数值离散方法，主要用于支撑相关领域
的基础算法研究与人才培养，也可用于CAE原型应用软件的快速开发验证。
